## 自动化：Azure VPS

### 启动 Azure VPS

``` bash
#!/bin/bash

# First log in with az login

# Variables - Replace with your desired values
resourceGroupName="myResourceGroup"
location="eastus" # Choose an Azure region like eastus, westus, etc.
vmName="myVM"
adminUsername="azureuser"
# You can generate a secure password or use SSH keys (recommended for Linux)
# For simplicity, this example uses a password. Replace 'YourSecurePassword123!'
adminPassword="YourSecurePassword123!"
vmImage="UbuntuLTS" # Specifies the latest Ubuntu LTS image
vmSize="Standard_B1s" # A small, cost-effective VM size

# Create Resource Group (if it doesn't exist)
echo "Creating resource group: $resourceGroupName"
az group create --name $resourceGroupName --location $location

# Create Virtual Machine
echo "Creating VM: $vmName"
az vm create \
  --resource-group $resourceGroupName \
  --name $vmName \
  --image $vmImage \
  --size $vmSize \
  --admin-username $adminUsername \
  --admin-password "$adminPassword" \
  --location $location \
  # --generate-ssh-keys # Uncomment this line to use SSH keys instead of a password

echo "VM $vmName created successfully."

# Optional: Get the public IP address
publicIp=$(az vm show --show-details --resource-group $resourceGroupName --name $vmName --query publicIps --output tsv)
echo "Public IP Address: $publicIp"

az vm stop \
  --resource-group $resourceGroupName \
  --name $vmName \
```
